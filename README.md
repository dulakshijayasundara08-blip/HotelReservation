# 🏨 Grand Horizon - Hotel Reservation System

A premium Hotel Management Web Application built using **Java EE (Jakarta Servlet & JSP)**, **MySQL**, and **Bootstrap 5**.

## ✨ Key Features
- **🔑 Glassmorphism Login & Registration System** (With Role-Based Access for Admin and Customer).
- **👤 Customer Dashboard:** View available luxury rooms, and book rooms with check-in/check-out dates.
- **🛠️ Admin Panel:** Add new rooms, view live reservation lists, and check real-time business statistics.
- **📊 Interactive Analytics:** Track Total Revenue, Active Bookings, and Available Rooms in real-time.
- **🔒 Secure Transactions:** Implemented Database Transaction Management (Commit & Rollback) for robust booking handling.

## 🛠️ Technologies Used
- **Backend:** Java (Jakarta Servlet, JSP, JDBC)
- **Frontend:** Bootstrap 5, HTML5, CSS3 (Custom Glassmorphism Design), Google Fonts
- **Database:** MySQL
- **Build Tool:** Maven
